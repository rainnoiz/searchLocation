การพัฒนาโปรแกรมนี้มีวัตถุประสงค์เพื่อศึกษาความเป็นไปได้ในการวิเคราะห์หาพิกัดที่ตั้งของร้านค้า คลังสินค้า หรือศูนย์กระจายสินค้าที่เหมาะสม ด้วยวิธีหาจุดศูนย์กลางแบบกริดเทคนิค (Grid Technique) และทำการเปรียบเทียบหาที่ตั้งที่มีระยะทางรวมหรือ ที่ตั้งที่ทำให้ประหยัดค่าใช้จ่ายในการขนส่งมากที่สุด ด้วยเทคนิคระยะทางร่วมกับค่าขนส่ง (Load-distance Technique) โดยการพัฒนาโปรแกรมนี้ได้นำเครื่องมือ Google Maps API มาประยุกต์ใช้เพื่อแสดงแผนที่ผ่าน Google Maps ซึ่งผลลัพธ์ที่ได้จากโปรแกรมนี้เป็นเพียงการคำนวณด้วยหลักการทางคณิตศาสตร์ ควรพิจารณาและตรวจสอบข้อมูลก่อนนำไปใช้จริง ดังนั้นหากเกิดความผิดพลาดประการใด ทางผู้จัดทำจะไม่รับผิดชอบใดๆ ทั้งสิ้น



สำหรับการทำวิจัยและการพัฒนาโปรแกรมนี้
จัดทำโดย น.ส.มัตติกา สิงโตสี และมีอาจารย์ที่ปรึกษาคือ ผศ.ดร.สรวิชญ์ เยาวสุวรรณไชย
