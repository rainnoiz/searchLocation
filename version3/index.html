โปรแกรมนี้พัฒนาขึ้นเพื่อศึกษาความเป็นไปได้ในการวิเคราะห์หาพื้นที่ตั้งของร้านค้า คลังสินค้า หรือศูนย์กระจายสินค้า ด้วยวิธีหาจุดศูนย์กลางแบบกริดเทคนิค (Grid Technique)
และทำการเปรียบเทียบหาที่ตั้งที่มีระยะทางรวมหรืออัตราค่าขนส่งต่อระยะทางน้อยที่สุด ด้วยเทคนิคระยะทางร่วมกับค่าขนส่ง (Load-distance Technique)
โดยการพัฒนาโปรแกรมนี้ได้นำเครื่องมือ Google Maps API มาประยุกต์ใช้เพื่อแสดงแผนที่ผ่าน Google Maps
